xsgens have awesome rates¶

xsgen is a tool for computing multi-group neutron cross-section, burnup, and multiplication factor (kinf) as a function of reactor state, which includes time/fluence, material properties, and reactor geometry.

We use a plugin-based system to allow a flexible approach with respect to underlying physics engines.

The source code can be found at the GitHub project site.

For a quick install from source, please clone from the official repo:

git clone git://github.com/bright-dev/xsgen.git
cd xsgen/
python setup.py install --user

Now you should be able to run it simply with:

xsgen
