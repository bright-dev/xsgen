A dictionary containing the following parameters:

The absolute time (relative to 0 days) that the transmutation occurs at.

Length of transmutation timestep - time relative to absolute time of last timestep.

The shell command for running ORIGEN.

Dictionary with neutron production and destruction rates, burnup, and transmutation results.
